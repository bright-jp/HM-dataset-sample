# H&M-dataset-sample

<h2>1001件のH&M商品サンプルデータセット</h2>

![H&M products dataset header](https://github.com/luminati-io/HM-dataset-sample/blob/main/HM-dataset.png)

1000件以上のレコードを含むH&M商品データセットのサンプルです。データセットは <b>Bright Data API</b> を使用して抽出されました。

<h2>H&M商品データセットに含まれるデータポイントの一部:</h2>

* ```category_tree```: 商品が属するカテゴリ階層  
* ```color```: 商品の色  
* ```country_code```: 商品の所在地または利用可能性を示す国コード  
* ```county_of_origin```: 商品の原産国  
* ```currency```: 商品価格が表示されている通貨  
* ```delivery```: 商品配送に関する情報  
* ```description```: 商品の詳細説明  
* ```domain```: 商品が掲載されているドメインまたはWebサイト  
* ```features```: 商品の特徴または特性  
* ```final_price```: 商品の現在価格  
* ```image_count```: 商品に関連付けられた画像の総数  
* ```image_urls```: 商品画像を指すURL  
* ```in_stock```: 商品が現在在庫ありかどうかを示します  
* ```initial_price```: 商品の元の価格または初期価格  
* ```main_image```: 商品に関連付けられたメイン画像のURLまたは識別子  
* ```manufacturer```: 商品の製造元  
* ```offers```: 商品に関する各種オファーまたはパッケージ情報  
* ```people_bought_together```: 現在の商品と一緒に購入されることが多い商品  
* ```product_name```: 商品名またはタイトル  
* ```reviews_count```: 商品に対する顧客レビューの総数  
* ```related_products```: 現在の商品に関連する他の商品  
* ```seller_name```: 商品を提供する販売者名  
* ```size```: 商品のサイズ  
* ```top_reviews```: 商品に関する注目レビューまたは上位の顧客レビュー  

ほかにも多数あります。

これは「H&M products」データセットから派生したサンプルサブセットであり、このデータセットには <b>410万件以上のレコード</b> が含まれます。

利用可能なデータセットのファイル形式: <b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じてファイルを .gz に圧縮できます</b>。

データセット配信タイプのオプション: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度: <b>1回のみ、毎日、毎週、毎月、四半期ごと、またはカスタム</b>。

抽出されたデータポイントに追加できるデータ enricH&Ment: <b>要リクエスト。</b>

<b>[H&M商品データセットのフル版を入手する](https://brightdata.jp/products/datasets/fashion/H&M)</b>。

<h2>H&M商品データセットのユースケースは何ですか？</h2>

<h3>1. 価格最適化</h3>
競合他社に対して、比較可能なH&M商品およびカテゴリを分析することで、価格戦略を策定し、ダイナミックプライシングモデルを作成します。

<h3>2. 商品在庫</h3>
H&M商品の在庫不足を特定し、特定アイテムの需要増加を検知し、消費者の間での新たなトレンドを明らかにします。

<h3>3. 市場戦略</h3>
H&Mデータセットを活用して市場戦略分析を実施し、主要トレンドと消費者の嗜好を特定します。
